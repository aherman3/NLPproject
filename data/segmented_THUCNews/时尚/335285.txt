心理 调频 第68 期 ： 你 的 情绪 “ 中暑 ” 了 吗 
 导语 ： 闷热 的 季节 ， 最 容易 有 无 名火 ， 即使 芝麻 大 的 事 ， 有人 也 会 怒不可遏 。 怒火 倘若 能 积极 地 加以 利用 ， 就是 一 种 强大 的 正面 能量 ， 而 选择 忍气吞声 ， 有时 反而 会 对 健康 产生 坏 影响 。 
 心理 调频 精彩 回顾 
 10 题 测试 ： 你 的 情绪 “ 中暑 ” 了 吗 ？ 
 1 、 你 邀请 的 客人 迟到 了 ， 而 准备 的 饭菜 都 凉 了 ， 你 会 ？ 
 A 、 装做 若无其事 的 样子 ， 但 今后 再 也 不 会 邀请 他们 了 。 ( 1分 ) 
 B 、 不时 挖苦 他们 。 ( 5分 ) 
 C 、 对 他们 不 像 以往 那么 热情 了 。 ( 3分 ) 
 2 、 对 好友 及时 提供 了 帮助 ， 但 他 ( 她 ) 对此 连 一 句 感谢 的话 也 不说 。 你 的 反应 ？ 
 A 、 责备 他 ( 她 ) 如此 不 懂 情理 。 ( 5分 ) 
 B 、 他 ( 她 ) 今后 应 明白 ， 朋友 该 如何 交往 相处 。 ( 3分 ) 
 C 、 感到 自己 受到 了 伤害 。 ( 1分 ) 
 3 、 当 别人 对 你 不 尊重 或 不 重视 时 ， 你 会 ？ 
 A 、 感到 十分 生气 。 ( 3分 ) 
 B 、 会 更 注意 自己 的 言行 。 ( 5分 ) 
 C 。 无所谓 。 ( 1分 )  
 4 、 排队 处 有人 不 遵守 秩序 ， 拼命 往前 挤 。 你 对此 的 态度 是 ？ 
 A 、 恼火 ， 大 喊别 插队 。 ( 5分 ) 
 B 、 仅仅 怒视 旁观 。 ( 3分 ) 
 C 、 万分 生气 但 闭口不言 。 ( 1分 ) 
 5 、 倘若 有人 对 你 很 不 礼貌 ， 你 会 ？ 
 A 、 顿时 感到 十分 生气 。 ( 5分 ) 
 B 、 我 会 退避三舍 。 ( 3分 ) 
 C 、 我 泰然 自若 。 ( 1分 ) 
 6 、 别人 能 与 你 打交道 和 商讨 问题 吗 ？ 
 A 、 谁 都 知道 我 常 有理 。 ( 5分 )           
 B 、 我 不 爱 与 人 争论 。 ( 1分 ) 
 C 、 善于 打交道 的 人才 能 与 我 商讨 问题 。 ( 3分 ) 
 7 、 和 老公 一起 外出 度假 ， 可 他 到达 目的 地后 就 埋怨 你 忘记 带 了 什么 。 你 对此 的 反应 ？ 
 A 、 顿时 发起 火来 ， 把 箱子 扔到 他 脚前 。 ( 5分 ) 
 B 、 让 他 感到 ， 他 大大 扫 了 我 的 兴 。 ( 3分 ) 
 C 、 我 休 我 的 假 。 ( 1分 ) 
 8 、 你 多次 告诉 母亲 别 再 寄 自己 童年 爱 吃 的 夹心 糖 了 ， 可 她 照 寄 不 误 。 你 的 反应 是 ： 
 A 、 她 越 老 脾气 越 怪 了 。 ( 3分 ) 
 B 、 不 再说 她 ， 随 她 去 。 ( 1分 ) 
 C 、 我 怏然 不悦 。 ( 5分 ) 
 9 、 什么 东西 最 使 你 发怒 ？ 
 A 、 不可 信赖 。 ( 1分 ) 
 B 、 谎言 。 ( 3分 ) 
 C 、 寻衅 和 攻击 。 ( 5分 ) 
 10 、 你 常 爱 发火 吗 ？ 
 A 、 从不 。 ( 1分 ) 
 B 、 一 年 两三 次 。 ( 3分 ) 
 C 、 动不动 就 发火 。 ( 5分 ) 
 测试 结果 ： 把 以上 每 道题 符合 自己 选项 后面 的 得 分加 起来 ， 对照 下面 的 结果 。 
 0 ～ 15 分    冷静型 
 乍看 起来 ， 冷静 的 好处 并 不少 。 这种 类型 的 人 对 令 其 发怒 的 情况 往往 不 去 争辩 ， 总是 隐藏 自己 的 不良 情绪 。 在 你 开始 像 往常 那样 压抑 你 的 怒火 之前 ， 你 应 真正 地 去 感受 一下 该 发怒 就 发怒 的 滋味 。 
 16 ～ 27 分    哽咽型 
 每当 你 一 受气 时 ， 就 会 激动 得 喉咙 哽住 说 不 出 话 来 ， 并 把 怒火 强咽 下去 。 这 对 你 无济于事 ， 你 应 把 受 伤害 的 感情 公开 地 表露 出来 。 这样 ， 当 你 发怒 时 ， 别人 就 会 重视 你 ， 而且 也 是 对 自己 的 爱护 。 
 28 ～ 38 分    抑制型 
 你 虽 十分 气愤 ， 但 却 不 发泄 出来 。 不过 你 一生气 、 一 发怒 ， 谁 都 能 一眼 看 出来 。 该 怒 不 发怒 ， 这样 做 对 你 自己 反而 有害 。 因为 压抑 怒火 不 发泄 出来 ， 反而 会 积怒 成疾 ， 并 变成 一 种 不良 的 癖好 。 
 38 分 以上    怒吼型 
 该 发怒 时 就 发怒 。 这 很 好 ！ 但 不 好的 是 你 常常 做 得 过分 ， 你 火冒三丈 往往 不是 别人 的 过错 。 今后 你 的 反应 要 更 理智 一些 。
